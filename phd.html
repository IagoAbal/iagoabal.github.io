static analysis and verification of highly configurable software

Software product line engineering is a cost-effective approach for developing and maintaining a family of software systems. Verification of software product lines is, however, a challenging task when dealing with large software families. The number of final products is often in the order of billions, and bugs can occur in any product as a result of feature interactions. This Ph.D. project aims to address the main difficulties in the applicability of verification techniques to highly configurable software, by using the Linux kernel as our primary subject of study.
