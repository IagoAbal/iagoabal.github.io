Types

data Z3_config

A configuration object used to initialize logical contexts.

data Z3_context

Logical context. This is the main Z3 data-structure.

data Z3_symbol

A Lisp-link symbol. It is used to name types, constants, and functions. A symbol can be created using string or integers.

data Z3_ast

Abstract syntax tree node. That is, the data-structure used in Z3 to represent terms, formulas and types.

data Z3_sort

A kind of AST used to represent types.

data Z3_func_decl

A kind of AST used to represent function symbols.

data Z3_app

A kind of AST used to represent constant and function declarations.

data Z3_pattern

A kind of AST used to represent pattern and multi-patterns used to guide quantifier instantiation.

data Z3_model

A model for the constraints asserted into the logical context.

type Z3_lbool = CInt

Lifted Boolean type: false, undefined, true.

z3_l_true, z3_l_undef, z3_l_false :: Z3_lbool

Values of lifted boolean type

type Z3_bool = CInt

Boolean type. It is just an alias for int.

z3_true, z3_false :: Z3_lbool

Z3_bool values

type Z3_string = CString

Z3 String type
